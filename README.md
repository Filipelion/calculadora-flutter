# Calculadora

Aplicativo de calculadora simples no estilo ios

## Como baixar

* Faça um fork do projeto (precisa ter uma conta no github)
* Abra um editor de texto
* Tenha o git
* git clone _link-do-seu-repositorio-git_  em diretorio
